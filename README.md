# dotfiles

These are my mac dotfiles that I use on the daily. I went for a dark blueberry theme and I would say it worked out pretty well!

![image3](https://i.imgur.com/H6vMWrb.jpeg)

## Things I configured
- Sketchybar: Custom bar for macos
- Spiceitfy: Dark blue colour scheme and other themings
- Neovim: Used my own colour scheme along with Mason for easier dev
- Kitty: Transparency and custom border settings

## Installation

Installation is pretty simple.

Go to your terminal and run:

```
bash install.sh
```

Using the [install.sh](https://github.com/kashsuks/dotfiles/blob/master/install.sh) file in this repository and then follow the additional instructions given by the command
